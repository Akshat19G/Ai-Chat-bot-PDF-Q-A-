📌 Description

Built a production-ready AI chatbot capable of answering questions from uploaded PDF documents using Retrieval-Augmented Generation (RAG). The system extracts document context, generates embeddings, stores them in a vector database, and provides accurate AI-generated responses through an interactive chat interface.

⚡ Features
PDF upload and parsing
Context-aware AI responses
Semantic search using vector embeddings
Conversational memory
Fast API-based backend
